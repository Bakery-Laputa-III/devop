0. clone仓库&进入仓库
    ```shell
    git clone git clone git://g.csail.mit.edu/xv6-labs-2020
    cd xv6-labs-2020
    ```
1. 更新apt仓库软件信息
    ```shell 
    sudo apt-get update -y
    ```
2. 安装gcc编译器
    ```shell
    sudo apt-get install gcc -y
    ```
3. 安装qemu(debian系列)
    ```shell
    sudo apt-get install qemu-system-misc -y
    ```
4. 编译/运行
    ```shell
    make qemu
    ```
